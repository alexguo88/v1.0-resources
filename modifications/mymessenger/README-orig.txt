My Messenger UseBB
-----------------------------

This Mod adds messenger functionality to UseBB. It currently works with UseBB .2.3. For install instructions, see docs/INSTALL.txt.

This software is released under the GNU-GPL. See docs/EULA.txt for more information.

For a list of the people involved, see docs/AUTHORS.txt.

This is a beta release. Bugs might (and probably do) exists. Should you encounter one, report it to the community forums or Mod Tracker at UseBB.net.

Enjoy!
